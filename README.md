# 智能试题练习系统

一个使用Flutter开发的智能试题练习应用，支持Excel题库导入和多种练习模式。

## 功能特性

- ✅ 多种题型支持（单选题、多选题、判断题）
- ✅ Excel题库导入功能
- ✅ 练习模式和考试模式
- ✅ 错题复习功能
- ✅ 学习进度统计
- ✅ 温暖柔和的用户界面

## 试题文件格式

### Excel题库格式

Excel题库文件应包含以下列（按顺序）：

| 列A | 列B | 列C | 列D | 列E | 列F |
|-----|-----|-----|-----|-----|-----|
| 题目ID | 题目类型 | 题目内容 | 选项内容 | 正确答案 | 解析 |

示例格式：
- 题目ID: 唯一标识符（可选，如不提供将自动生成）
- 题目类型: 单选题/多选题/判断题
- 题目内容: 题目的具体内容
- 选项内容: 选项格式如 "A:选项A\|B:选项B\|C:选项C\|D:选项D"
- 正确答案: 单选题如"A"，多选题如"ABC"，判断题如"A"（正确）或"B"（错误）
- 解析: 答案解释

### 内置题库

应用还支持JSON格式的内置题库，位于 `assets/questions.json`。

## 构建与部署

### 本地构建

1. 确保已安装Flutter SDK
2. 安装依赖：`flutter pub get`
3. 构建APK：`flutter build apk --release`

### GitHub Actions自动构建

本项目配置了GitHub Actions自动构建工作流：

1. **普通构建**: 推送代码到main/master分支时自动构建
2. **发布构建**: 创建标签时构建带签名的发布版APK

#### 配置发布构建（可选）

如需使用发布构建，需要在GitHub仓库的Secrets中设置以下变量：

- `SIGNING_KEY`: Base64编码的签名密钥
- `KEY_ALIAS`: 密钥别名
- `KEY_PASSWORD`: 密钥密码
- `KEY_STORE_PASSWORD`: 密钥库存储密码

## 使用说明

### 题库导入

1. 将Excel题库文件放入设备的"题库"文件夹
2. 应用会自动检测并导入题库文件
3. 题库将被转换为SQLite数据库格式以提高性能

### 应用功能

- **练习模式**: 随机练习题目
- **错题复习**: 专门复习答错的题目
- **题库统计**: 查看题库信息
- **学习进度**: 查看学习统计信息
- **模拟考试**: 完整的考试体验
